8. Тригонометрические уравнения

Общие и частные решения, методы решения уравнений с тригонометрическими функциями

Что такое тригонометрическое уравнение?

Тригонометрическое уравнение — это уравнение, содержащее одну или несколько тригонометрических функций (sin, cos, tg, ctg) от переменной.

Основные виды и решения

  • \( \sin x = a \Rightarrow x = (-1)^n \arcsin a + \pi n \)
  • \( \cos x = a \Rightarrow x = \pm \arccos a + 2\pi n \)
  • \( tg x = a \Rightarrow x = arctg a + \pi n \)
  • \( ctg x = a \Rightarrow x = arcctg a + \pi n \)

Примеры уравнений

  • \( \sin x = \frac{1}{2} \Rightarrow x = \frac{\pi}{6} + 2\pi n, \frac{5\pi}{6} + 2\pi n \)
  • \( \cos x = -\frac{\sqrt{2}}{2} \Rightarrow x = \frac{3\pi}{4} + 2\pi n, \frac{5\pi}{4} + 2\pi n \)

Методы решения

  • Использование формул обратных функций
  • Использование тригонометрических тождеств
  • Приведение уравнений к квадратным или однородным
  • Подстановка, разложение и методы замены

Общий подход

  1. Упростить уравнение с помощью тождеств
  2. Найти общее решение элементарного уравнения
  3. Учесть область определения (ОДЗ)

Практическое задание

1. Найдите общее решение уравнений

  1. \( \sin x = \frac{1}{2} \)
  2. \( \cos x = -\frac{1}{2} \)
  3. \( tg x = 1 \)

2. Найдите корни на интервале \( [0; 2\pi] \)

  1. \( \sin x = -\frac{\sqrt{2}}{2} \)
  2. \( tg x = \sqrt{3} \)

3. Упростите уравнение и решите

  1. \( 2\sin^2 x - 1 = 0 \)
  2. \( \cos^2 x - \sin^2 x = 0 \)

Тест: тригонометрические уравнения

1. Чему равен корень уравнения \( \sin x = 1 \) на интервале \( [0; 2\pi] \)?

2. Сколько корней имеет уравнение \( \cos x = 0 \) на \( [0; 2\pi] \)?

3. Как решается уравнение \( tg x = 1 \)?
