7. Обратные тригонометрические функции

Теория, примеры и тренировка по arcsin, arccos и arctg

Что такое обратные тригонометрические функции?

Обратные тригонометрические функции — это функции, которые возвращают угол по заданному значению тригонометрической функции. Они обозначаются как:

  • arcsin(x) — обратная к sin(x)
  • arccos(x) — обратная к cos(x)
  • arctg(x) — обратная к tg(x)

Области определения и значений

  • arcsin(x): x ∈ [–1, 1], результат ∈ [–π/2, π/2]
  • arccos(x): x ∈ [–1, 1], результат ∈ [0, π]
  • arctg(x): x ∈ ℝ, результат ∈ (–π/2, π/2)

Примеры

  • arcsin(1/2) = π/6
  • arccos(0) = π/2
  • arctg(1) = π/4

Графики

Графики обратных функций можно получить отражением обычных графиков sin(x), cos(x), tg(x) относительно прямой y = x, но только на тех участках, где функции являются обратимыми (монотонны).

Важно помнить: arcsin(x) и arccos(x) дают только главное значение угла. Остальные можно получить с помощью общих формул:

\[ \sin x = a \Rightarrow x = (-1)^n \cdot \arcsin a + \pi n \]

\[ \cos x = a \Rightarrow x = \pm \arccos a + 2\pi n \]

Практическое задание

1. Вычислите значения

  1. arcsin(0)
  2. arccos(1)
  3. arctg(–1)

2. Найдите все решения уравнения

sin(x) = 1/2, x ∈ [0, 2π]

Используйте arcsin и свойства периодичности синуса.

3. Постройте график функции

На миллиметровке или в Desmos постройте графики функций:

  • y = arcsin(x)
  • y = arccos(x)
  • y = arctg(x)
