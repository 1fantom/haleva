Функции y = tg(x) и y = ctg(x)

Свойства, графики, анимация и практика

Определения и тождества

tg x = sin x / cos x    |    ctg x = cos x / sin x

Графическое представление функций

Мини-анимация tg x

Точка движется по графику y = tg x (диапазон –π/2 … π/2)

Основные тождества

1 + tg²x = 1 / cos²x
1 + ctg²x = 1 / sin²x

Свойства

  • Период π, обе функции нечётные
  • tg x не определена при x = π/2 + πn; ctg x – при x = πn
  • Вертикальные асимптоты совпадают с точками, где функции не определены

Практическое задание

1. Таблица значений

Заполните таблицу, вычислив tg x и ctg x:

2. Упрощение выражений

  1. (1 + tg²x)(1 – sin²x)
  2. tg x · ctg x
  3. tg²x – 1
  4. ctg²x – cos²x / sin²x

3. Решите уравнения

  1. tg x = √3, 0 ≤ x < 2π
  2. ctg x = –1, 0 ≤ x < 2π
  3. tg 2x = 1, 0 ≤ x < π

4. Исследуйте графики
